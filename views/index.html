URL Shortener by Justin Thompson

Takes url, returns JSON of shortened url that redirects to original url

Example creation

https://jt-urlshortener.herokuapp.com/new/https://www.google.com
https://jt-urlshortener.herokuapp.com/new/http://foo.com:80
Example creation output:
{ "original_url":"http://foo.com:80", "short_url":"https://jt-urlshortener.herokuapp.com/8170" } Usage:
https://jt-urlshortener.herokuapp.com/2871
Will redirect to: https://google.com/
